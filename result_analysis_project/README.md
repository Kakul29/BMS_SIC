# Result Analysis System

## Overview

This project analyzes student assessment results and generates PDF reports for both students and examiners.

## Features

* Supports MCSR and MCMR questions
* Subject-wise analysis (OOP, DBMS, DSA, Pandas)
* Difficulty-wise analysis (Easy, Intermediate, Complex)
* Student performance reports
* Examiner summary report
* Time-based analytics
* Grade calculation
* Ranking system

## Technologies Used

* Python
* Pandas
* OpenPyXL
* ReportLab
* SQL

## Output

* Individual Student PDF Reports
* Examiner Summary PDF
* Subject-wise Performance Analysis
* Difficulty-wise Performance Analysis
